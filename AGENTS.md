# Elefante Project Instructions

Read `START_HERE.md`, `specs/elefante-project-toolchain.md`, and `ELEFANTE_BRAND_SETUP.md` before making product, architecture, branding, or implementation decisions.

Elefante is the current product name. The retired placeholder may still appear in historical Git commits and the temporary GitHub remote.

Composer remains the PHP dependency authority. Elefante owns the executable environment around Composer.

The complete vision includes a vertically integrated local PHP platform, first class multi project and Git worktree isolation, shared services with isolated data, process supervision, local routing, trusted TLS, database workflows, tunneling, and eventual production delivery.

Implementation order is Apple Silicon macOS, Linux, then Windows. Go is the primary implementation language and the intended distribution is a single native binary.

Do not rename or transfer the external GitHub repository, push commits, publish packages, register accounts, or deploy services without explicit approval.
